#  Unemployment Analysis in India

## Project Description

This project analyzes unemployment data in India using Python. The goal is to understand unemployment trends by cleaning the data, performing basic analysis, and creating visualizations to gain useful insights.

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

##  Dataset

The project uses unemployment datasets containing information such as:

- Region
- Date
- Unemployment Rate
- Estimated Employment
- Labour Participation Rate

##  Project Steps

- Imported the required libraries.
- Loaded the datasets.
- Checked and cleaned the data.
- Handled missing values.
- Performed Exploratory Data Analysis (EDA).
- Calculated the average unemployment rate.
- Compared unemployment rates across different states.
- Created charts to visualize the data.
##  Results

- Analyzed unemployment trends in different states.
- Identified states with high and low unemployment rates.
- Created simple visualizations to better understand the data.
- Gained useful insights from the unemployment dataset.
